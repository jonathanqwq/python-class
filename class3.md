# Python3的介绍＋入门--第二讲：print

> 欢迎您，尊敬的python开发者！

经过前几节课的学习，我们已经把准备工作全部做好了。接下来，我们就可以学到我们python中的第一个代码```print```

一、单行输出
```python
print("Hello Python")  
```
运行结果：  
```Hello Python```


二、多行输出
```python
print("床前明月光，")
print("疑似地上霜。")
```
运行结果： 
```
床前明月光，
疑似地上霜。
```
